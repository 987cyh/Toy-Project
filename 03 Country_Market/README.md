# 나라장터 데이터 수집 및 서면 보고

https://www.g2b.go.kr/

Version_1
1. API를 통한 입찰공고정보, 사전규격, 낙찰내역 수집 및 저장 (로컬)
2. 보고용 엑셀 파일 생성 및 SLACK 채널로 자동 전송
3. 파일 확인 후 서면 보고 수행

Version_2
1. API를 통한 데이터 수집 및 저장 (MYSQL DB)
2. 보고용 엑셀 파일 생성 및 slack 메신저를 통한 채널에 전송
3. 그라파나를 활용한 모니터링

Version_3
1. AirFlow를 통한 자동화 프로세스 구축
2. 보고용 엑셀 파일 생성 및 slack 메신저를 통한 채널에 전송
3. 그라파나를 활용한 모니터링
